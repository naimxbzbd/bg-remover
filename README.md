# 🎯 AI Background Remover by Naim Xbz

A powerful, browser-based background removal tool built with pure HTML, CSS, and JavaScript. Remove backgrounds from images instantly with AI - all processing happens locally in your browser!

## ✨ Features

- 🤖 **AI-Powered** - Advanced background removal using @imgly/background-removal
- 🚀 **Instant Processing** - Remove backgrounds in seconds
- 📁 **Multiple Upload Options**:
  - Drag & Drop
  - File Browser
  - Image URL
  - Sample Images
- 🎨 **Background Options**:
  - Transparent
  - Solid Colors (White, Black, Blue, Green)
  - Custom Colors
  - Gradients
- ✏️ **Manual Refinement Tools**:
  - Restore Brush
  - Erase Brush
  - Adjustable Brush Size
- 📊 **Comparison Views**:
  - Split View
  - Original Only
  - Removed Only
- 💾 **Export Options**:
  - PNG (with transparency)
  - JPG
  - Copy to Clipboard
- 🔒 **Privacy First** - All processing done locally
- 📱 **Fully Responsive** - Works on all devices
